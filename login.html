个人日记应用

使用GitHub存储您的日记内容

安全提示：此令牌将授予对您指定仓库的完全访问权限。请仅用于个人使用，并且不要在公开设备上保存此令牌。

如何获取GitHub个人访问令牌:

  1. 访问 GitHub Token设置
  2. 点击 "Generate new token"
  3. 设置一个有意义的描述（例如："日记应用访问令牌"）
  4. 选择令牌的有效期（推荐设置为永不过期）
  5. 选择权限范围：
    • repo（完全访问私有仓库）或者
    • public_repo（如果使用公开仓库）
  6. 点击"Generate token"并复制生成的令牌
  7. 返回此页面，粘贴令牌并点击"确认并继续"
您的日记将安全存储在您的GitHub仓库中
